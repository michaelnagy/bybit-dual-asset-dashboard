# Bybit Dual Asset Overlay Extension

This folder contains the Chrome extension runtime skeleton for the Bybit Dual Asset overlay.

## Load Unpacked

1. Open `chrome://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select this `extension/` folder

## Current Status

- `Part 2` provides the overlay shell only
- It mounts on the Bybit Dual Asset orders page
- It supports drag, resize, minimize, and persisted window state
- Real Bybit data wiring is planned for `Part 3`

## Target Page

- `https://www.bybit.com/user/assets/order/financial/financial-dual-asset-orders/dual-asset-order`
